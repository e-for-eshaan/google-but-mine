# Next.js Project Setup

## Prerequisites
- Node.js (v18+)
- PNPM

## Setup Instructions

1. Clone the repository
2. Install dependencies:
```bash
pnpm install
```

3. Run development server:
```bash
pnpm dev
```

## Project Structure
- `src/`: Source code
- `src/app/`: Next.js app router
- `src/stores/`: Zustand state management
- `src/server/`: tRPC server-side logic

## Key Technologies
- Next.js 13+
- TypeScript
- Tailwind CSS
- tRPC
- Zustand
- ESLint
- Prettier
